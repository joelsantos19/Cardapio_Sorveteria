# Sorveteria Jenkins 🍦

Descrição do cardápio digital moderno e funcional...

## ✨ Funcionalidades
- Tela de Boas-Vindas
- Seleção de Mesa
- Cardápio Interativo...

## 🛠️ Tecnologias Utilizadas
- Flutter, Dart e Google Fonts.

## 🚀 Como Executar o Projeto (Getting Started)
1. Instalar Flutter.
2. `flutter pub get`.
3. `flutter run`.
